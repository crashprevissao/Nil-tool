# ESP32 Hacking Device

Dispositivo multifuncional para testes de segurança.

## Hardware
- ESP32 DevKit
- NRF24L01
- CC1101
- OLED 0.96"
- 4 Botões
- SD Card (opcional)

## Funcionalidades
- Câmera: Congelar/Desconectar
- Clonagem: Copiar/Reproduzir sinais
- Drone: Jamming/Controle
- Bluetooth: Desautenticar/Conectar
- Wi-Fi: Deauth/Scan
- Configurações: Salvar em Memória ou SD

## Compilação
Automaticamente via GitHub Actions ou local com PlatformIO.
